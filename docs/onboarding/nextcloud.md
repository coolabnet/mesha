# Nextcloud — Community File Storage

## What is Nextcloud?

Nextcloud is a file storage and sharing service that runs on the community's own server. Think of it like Google Drive or Dropbox, except it belongs to the community — your files never leave the neighbourhood.

---

## How to Access Nextcloud

Nextcloud works on your phone, tablet, or computer. You do not need an internet connection — it works over the community WiFi network.

**Address:** Open a browser and go to:

```
https://nuvem.bairro.local
```

You must be connected to the **community WiFi network** for this to work. It will not open if you are using mobile data or a different WiFi network.

> **Tip:** The first time you visit, your browser may show a security warning about the certificate. This is normal for local community servers — click **"Advanced"** and then **"Accept"** (or **"Proceed"**) to continue. You will only see this once.

> **If the page does not open:** make sure you are connected to the community WiFi. If it still does not open, contact the community maintainer.

---

## Creating Your Account

If the community allows self-registration, you will see a link to create an account on the login page.

1. Click **"Create account"** or **"Register"** on the login page
2. Choose a username (your name or a nickname — keep it simple, no spaces)
3. Choose a password you will remember — make it at least 8 characters
4. Click **"Create account"**

If there is no registration link, ask the community maintainer to create an account for you. They will give you a username and a temporary password to change when you first log in.

## Uploading and Sharing Files

### To upload a file from your computer or phone:

1. Log in to Nextcloud
2. Click the **"+"** button (top of the page or near the file list)
3. Select **"Upload file"**
4. Choose the file from your device
5. Wait for the upload to finish — a progress bar will show the status

### To share a file with someone:

1. Find the file you want to share in your file list
2. Click the **share icon** (a dot with lines, or a person icon) next to the file name
3. Type the username of the person you want to share with, then press Enter
4. You can also create a **share link** to send via WhatsApp or Telegram — click "Share link" and copy the address

### To create a folder:

1. Click the **"+"** button
2. Select **"New folder"**
3. Type a name and press Enter

## Mobile App Setup (Android / iPhone)

Using the Nextcloud app on your phone makes it much easier to upload photos and access files.

### Android

1. Open the **Play Store** on your phone
2. Search for **"Nextcloud"**
3. Install the app by Nextcloud GmbH
4. Open the app
5. Tap **"Log in"**
6. For the server address, type: `https://nuvem.bairro.local`
7. Tap **"Next"**, then enter your username and password
8. Tap **"Log in"** to confirm

### iPhone (iOS)

1. Open the **App Store**
2. Search for **"Nextcloud"**
3. Install the app by Nextcloud GmbH
4. Open the app
5. Tap **"Log in"**
6. For the server address, type: `https://nuvem.bairro.local`
7. Tap **"Next"**, then enter your username and password
8. Tap **"Log in"** to confirm

> **Note:** You need to be on the community WiFi network when you first set up the app. After setup, the app will only sync when you are on the community WiFi.

The maintainer can reset your password or help you get access.
